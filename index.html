Hello and welcome to my website! If you are able to view this website, that means my Raspberry Pi is up and running. For those who aren't familiar with the device, a Raspberry Pi is a credit card sized single-board computer which allows users to experiment and learn more about the world of computers at an inexpensive cost. With the Raspberry Pi, I created a web server to host this website I created using HTML, CSS, and Javascript.
